Class: SearchController

SearchController(containerId, movies, onScreenChange, onDataChange)

контроллер дя управления навигацией – отрисовка, переключение экранов статистики и списка, выбор фильтров.

Constructor

new SearchController(containerId, movies, onScreenChange, onDataChange)

Parameters:
Name Type Description
containerId String – id родительского контенйера для рендеринга.
movies Array – список фильмов.
onScreenChange Func обратотчик,который вызывается при переключении на экран статистики и обратно.
onDataChange Func – обработчик, который вызывается при изменении данных в списке фильмов.
